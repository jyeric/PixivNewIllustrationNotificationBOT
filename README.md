# PixivNewIllustrationNotificationBOT

Pixiv关注的人上传新插图通知机器人

本项目基于[pixivpy](https://github.com/upbit/pixivpy) 

### 安装/install

```
pip install pixivpy --upgrade
git clone https://github.com/jyeric/PixivNewIllustrationNotificationBOT.git
python index.py
```
